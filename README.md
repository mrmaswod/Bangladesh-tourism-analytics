# Bangladesh Tourism Analytics: Insights and Implications

**Goal**  
Turn fragmented Bangladesh tourism data into decisions. Python (Pandas, Matplotlib) for analysis; Power BI for interactive dashboards.

**Key Insights**  
- Domestic spend dominates → prioritize regional hospitality and transport.  
- Seasonality: monsoon dip, Nov–Feb peak → plan campaigns and capacity.  
- High-potential circuits: Sonargaon, Paharpur, Mahasthangarh → product development.
